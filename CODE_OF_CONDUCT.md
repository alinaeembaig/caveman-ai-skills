# Code of Conduct

Caveman AI Skills is intended to be a helpful, professional, and inclusive open-source project.

## Expected behavior

- Be respectful and constructive.
- Keep feedback specific and useful.
- Avoid personal attacks or harassment.
- Respect different tools, languages, frameworks, and workflows.
- Keep contributions project-safe and stack-agnostic.

## Unacceptable behavior

- Harassment, insults, or discriminatory language
- Publishing private information without permission
- Submitting secrets, credentials, or proprietary code without authorization
- Intentionally adding harmful or destructive instructions

## Enforcement

Repository maintainers may remove comments, close issues, reject pull requests, or restrict participation when behavior violates this code of conduct.
